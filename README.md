# business-listing
A basic application for business listing

[API](https://api.jsonbin.io/b/625eb6da80883c3054e3915e/13).
[Design](https://imgur.com/a/7NP228q).

The application with 2 views. The list of the business (also home page) 
contains all the businesses - listed with name and description. 
By clicking any of them, the user will be taken to another view which will display more information about the selected business.

Each business view has its own URL/route and contains the following information:
   - Business address
   - Contact information - name and phone number
   - Nearby places - a list of other businesses with the same country as the selected business
